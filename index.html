Published Packages

Open-source Flutter plugins I authored and published on pub.dev.

eli_ble_gatt_server pub.dev

v0.2.0 · MIT License

Flutter plugin that turns a device into a BLE GATT Server (peripheral), advertising custom services and communicating with Bluetooth Low Energy clients. Supports Android (API 21+) and iOS (13+), custom Service/Characteristic UUIDs, READ/WRITE/NOTIFY operations with real-time event streams, broadcasting to all connected clients via sendMessage(), and an Android foreground service.

View on pub.dev View on GitHub

abi_beacon pub.dev

v0.1.0 · MIT License

Android Flutter plugin for iBeacon background monitoring that keeps scanning even when the app is closed or the screen is off, using a persistent foreground service. Provides region entry/exit detection, ranging (RSSI & distance estimates), runtime permission handling, and — uniquely — management of manufacturer-specific battery restrictions (Xiaomi, Huawei, Samsung, OnePlus) with deep links to OEM settings and Doze exemption status. (Android API 24+).
